# Facebook App Ads

Updated: Apr 14, 2026

Mobile app ads help you get more people to install and engage with your app. This guide is for software developers who are setting up an app for app ads. It contains technical instructions on how to set up and help optimize delivery of your mobile app ads. If you aren’t already familiar with app ads, you can start by reading the background information at [Business Help Center: About App Ads⁠](https://www.facebook.com/business/help/1471413626484885?ref=fbb_app_installs). Then, come back here for implementation details.

If you’re not a developer, we recommend that you get help to complete your app setup:

* Get a developer to help, or
* If you or your organization are working with a [Facebook Marketing Partner⁠](https://www.facebook.com/business/marketing-partners/), your partner can complete all the technical setup you need.

Once app setup is complete, you can create and run app ads in [Ads Manager⁠](https://www.facebook.com/ads/create/) or via [API](https://developers.facebook.com/documentation/ads-commerce/marketing-api/mobile-app-ads).
