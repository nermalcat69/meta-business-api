# Looking for Help?

Updated: Jun 16, 2026

## Support

If you’re stuck on something with WhatsApp Flows, start here to get help. Use this page to share feedback.

### [Direct support](https://developers.facebook.com/documentation/business-messaging/whatsapp/support)

If you are a managed partner, please select *WABiz: Cloud API / WABiz: On Prem* and then select request type as *WA Flows*.

### [Developer community](https://developers.facebook.com/community?sort=trending&category=766772797555412&tag=870321727692773)

You can check [current developer questions here](https://developers.facebook.com/community?sort=trending&category=766772797555412&tag=870321727692773). When asking a question please use *Category: WhatsApp Business API > Product: WhatsApp Flows*.

### [Stack Overflow⁠](https://stackoverflow.com/questions/tagged/whatsapp-flows)

You can check current questions about Flows on Stack Overflow.

## Status

Check on the [status⁠](https://metastatus.com/whatsapp-business-api) of WhatsApp and WhatsApp Flows using the external status tool.
