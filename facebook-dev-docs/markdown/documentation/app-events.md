# Meta App Event Tracking

Updated: Sep 26, 2023

Meta App Events allows your app or web page to track events, such as a person installing your app or completing a purchase.

For information on tracking App Events for Business Messaging, please see the [**App Events API for Business Messaging**](https://developers.facebook.com/documentation/ads-commerce/conversions-api/business-messaging) in our [Messenger Platform documentation](https://developers.facebook.com/documentation/business-messaging/messenger-platform).

## Common Uses

* [Track events in Android apps](https://developers.facebook.com/documentation/app-events/getting-started-app-events-android)
* [Track events in iOS apps](https://developers.facebook.com/documentation/app-events/getting-started-app-events-ios)
