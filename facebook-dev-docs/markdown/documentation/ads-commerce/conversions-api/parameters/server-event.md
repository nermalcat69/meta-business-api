# Main Body Parameters

Updated: Nov 10, 2021

| Parameter | Description |
| --- | --- |
| `data`  array<object> | **Required.**  An array of server event objects. See [Server Event Parameters](https://developers.facebook.com/documentation/ads-commerce/conversions-api/parameters/server-event) for more information. |
| `test_event_code`  string | **Optional.**  Code used to verify that your server events are received correctly by Facebook. Use this code to test your server events in the Test Events feature in Events Manager. See [Test Events Tool](https://developers.facebook.com/documentation/ads-commerce/conversions-api/using-the-api#testEvents) for an example. |
