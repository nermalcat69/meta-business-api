# Guides

Updated: Jun 29, 2022

Use these guides to help you perform common actions with your catalog setup.

### [Product Categories](https://developers.facebook.com/documentation/ads-commerce/commerce-platform/catalog/categories)

Learn more about the catalog fields that describe the type or category of commerce products in your listing.

### [Product variants](https://developers.facebook.com/documentation/ads-commerce/commerce-platform/catalog/variants)

Learn more about product variants and how to use them in your product catalog.

### [Inventory](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/inventory)

Learn how to manage your catalog inventory.

### [Feed API](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/feed-api)

Learn how to upload inventory, handle feed upload errors, and download a product feed errors report.

### [Product deep links](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/product-deep-links)

Learn more how to use deep link information in your feed.

### [Scheduled Feeds](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/scheduled-feeds)

Learn how to schedule data feed uploads and fetches.

### [Microdata tags](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/microdata-tags)

Learn how to use OpenGraph, Schema.org, and JSON-LD for Schema.org protocols to set microdata tags in your catalog feed file.

### [Metadata tags](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/metadata-tags)

Learn how to set metadata tags in your product feed files.

### [Localized catalogs](https://developers.facebook.com/documentation/ads-commerce/catalog/guides/localized-catalog)

Learn how to set up your catalog for localized shopping with multiple languages and countries. See also [Localized Catalog Setup](https://developers.facebook.com/documentation/ads-commerce/catalog/localized-catalog/localized-catalog-setup), [Localized Catalog for Advantage+ Catalog Ads](https://developers.facebook.com/docs/marketing-api/catalog/localized-catalog/localized-catalog-da) and [Localized Catalog for Instagram Shopping](https://developers.facebook.com/docs/marketing-api/catalog/localized-catalog/localized-catalog-ig).
