# Conversions API Gateway Data Pipeline

Updated: Feb 2, 2025

The Conversions API Gateway data pipeline will only allow Meta Pixel as a data source, and Meta Conversions API as a data destination. You can manage your data pipelines in the Data pipeline page, including creating and deleting pipelines.

## Pipeline Creation

Click the **Create data pipeline** button on the top right drop down menu on the Data pipelines page.

![Create data pipeline dialog asking which pipeline type to create, with the Conversions API Gateway pipeline option selected.](https://scontent.fdel27-7.fna.fbcdn.net/v/t39.2365-6/473246593_3450389201760307_6154661319874312511_n.png?_nc_cat=101&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=Zn9TiiDaZ9kQ7kNvwGfqDpP&_nc_oc=AdouHg-WGWeg7A7WNUYYvp4Gd91byrJVE_qP1dQ2y_nX92psHjJwGKDabBB6w15edWOKLKVN7LWrHamp_wO-bEWi&_nc_zt=14&_nc_ht=scontent.fdel27-7.fna&_nc_gid=GC6V5ZH_ERTTtnM17Lo7wA&_nc_ss=7b289&oh=00_AQABWWjdogvih_S6y_9hyKoZFx_sHFALo7C6U-rhQJwOWg&oe=6A609773)

After selecting the **Conversions API Gateway pipeline** option, you can create a Conversions API Gateway pipeline by connecting a Meta Pixel selected from the pixel connection flow.

## Deleting a Data Pipeline

Click the name of the Conversions API Gateway data pipeline and go to the pipeline details page. Then find the **Actions** drop down button on the top right corner, and click **Delete pipeline**.

![Conversions API Gateway pipeline details page with the Actions drop down open, showing Disable pipeline and Delete pipeline options.](https://scontent.fdel27-1.fna.fbcdn.net/v/t39.2365-6/473728679_1287048072569682_1217971639256397142_n.png?_nc_cat=110&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=c6uBU1Z09uEQ7kNvwFa4uUI&_nc_oc=Adquqx9MrkZv7lblyvEZQPBn3Ft-GpzVV4b_DB3yho7I5-_h1npgkM0_vgtJFmRhrdZEwd0ic9lQM7QnCHMO9y-I&_nc_zt=14&_nc_ht=scontent.fdel27-1.fna&_nc_gid=GC6V5ZH_ERTTtnM17Lo7wA&_nc_ss=7b289&oh=00_AQBo6gwXrRUCZM3T7WKx8_TXrL4Oo81OVfKAbWwRDZraTg&oe=6A608971)

You can also delete a Conversions API Gateway data pipeline from either the **Data source** or **Data destination** tab. Click the Meta Pixel data source or data destination, and find the **Actions** drop down button on the top right corner, and click **Disconnect pipeline**.

![Meta Pixel data source panel with the Actions drop down open, showing Disable data source and Disconnect pipeline options.](https://scontent.fdel27-6.fna.fbcdn.net/v/t39.2365-6/474068717_1516019959085420_1291549429887024858_n.png?_nc_cat=107&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=1IdnMAPHKuMQ7kNvwENRPKZ&_nc_oc=AdqIqzG_3OSD_kc2h9-sB2jWvTJX991FdLetYuQ5ncjqILEdcwaF45c2sexS1t5SDORh73yw-MzNrKk9aNv-rrpW&_nc_zt=14&_nc_ht=scontent.fdel27-6.fna&_nc_gid=GC6V5ZH_ERTTtnM17Lo7wA&_nc_ss=7b289&oh=00_AQA6-sau7fBOaFAu62nwk64UKh36D3TmZKokTGGwUMKPVA&oe=6A607921)
