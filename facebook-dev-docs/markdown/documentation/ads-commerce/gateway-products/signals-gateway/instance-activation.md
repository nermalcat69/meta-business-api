# Signals Gateway: Setup Guide

Updated: Apr 30, 2026

Use the following steps to set up the Signals Gateway.

Please ensure you have reviewed and completed the [prerequisites for Signals Gateway](https://developers.facebook.com/documentation/ads-commerce/gateway-products/configuration) before you go through this setup guide.

## Deployment Option 1: Meta Events Manager

### Setup options

* Partner-hosted
  * Stape
  * Madgicx
  * DataHash
  * Birch
* Self-hosted
  * AWS ECS Express
  * GCP GKE
  * GCP Cloud Run

To begin the setup process in Meta Events Manager, navigate to the **Settings** for a selected data source. Under “Set up with Gateway Products”, select the “Signals Gateway” option and click on **Get started** and follow the setup steps.

There are multiple onboarding options for Signals Gateway. To use a shared infrastructure that is managed by an external partner, select the “Host with a partner” option.

To create a dedicated instance on your own cloud account, select “Host Yourself” and follow the setup steps to create a cloud instance.

## Deployment Option 2: Cloud Template

### Setup options

* Self-hosted
  * AWS ECS Express
  * GCP GKE
  * GCP Cloud Run

Navigate to the [Signals Gateway microsite⁠](https://www.facebook.com/business/m/signalsgateway/), click on the **Get started** button and follow the rest of the setup steps.
