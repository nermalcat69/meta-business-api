# Instance Activation

Updated: Feb 2, 2025

Access the admin domain (specified during the stack creation process) in the browser. Log in with the email address and password provided during stack configuration.

Please complete the initial setup phase of Signals Gateway using Google Chrome. You will need to link the Signals Gateway instance to Meta in order to share data with Meta, and some browsers create errors during this process. Please note that this is not required for usage of Signals Gateway. Please make sure ad-blockers are disabled as well.

## 3. Choose a Name

Provide the name for your organization. If you have a Meta Business Account, we recommend using the name of the business as shown in the corresponding Business Account. Meta will use this name to attribute event traffic to your organization. If telemetry is enabled, this name will also identify all telemetry (monitoring) data sent by the Signals Gateway instance to Meta.
