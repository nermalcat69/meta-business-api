# Conversions API Gateway: Instance Activation

Updated: Feb 2, 2025

Access the admin domain (specified during the stack creation process) in the browser. Login with the email address and password provided during stack configuration.

Please complete the initial setup phase of Conversions API Gateway using Google Chrome. Please note that you will need to link the Conversions API Gateway instance to Meta, and some browsers create errors during this process. Please make sure ad-blockers are disabled as well.

Once you’ve logged in successfully as an admin user, you’ll be guided through the final configuration steps of the Conversions API Gateway.

### 1. Setup Overview

From the Overview screen, click **Next** to continue the setup process.

![Conversions API Gateway setup wizard on the Overview step with a four-step sidebar and a Next button.](https://scontent.fdel27-8.fna.fbcdn.net/v/t39.2365-6/386607301_1938993873142450_4229691685485778199_n.png?_nc_cat=110&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=i2wn5awCdAwQ7kNvwFkyLQo&_nc_oc=AdpFwUwUVNZ4H_CA7h5oHW7_cF43PnYq4PDPI4OU-xRslUTHd4OddU-UQiIaUnVSHBNe-WQVZx5jDQ4cqKdwVxSy&_nc_zt=14&_nc_ht=scontent.fdel27-8.fna&_nc_gid=8JW2Q1mCYFbKMdcyEX47vQ&_nc_ss=7b289&oh=00_AQCEWwFXlPXfuY71RjxKDKrn-Cy0dLBzxXHfZDBGXBUX9A&oe=6A606B21)

### 2. Choose a Name

Provide the name for your organization; we recommend using the name of the business as it is shown in the corresponding Meta Business Account. Meta will use this name to attribute event traffic to your organization. If telemetry is enabled, this name will also identify all telemetry (monitoring) data sent by the Conversions API Gateway instance to Meta.

Click **Next** to continue.

![Choose name step of the setup wizard with an Organization name field and Back and Next buttons.](https://scontent.fdel27-8.fna.fbcdn.net/v/t39.2365-6/386644545_1636406046850241_2506754562293153332_n.png?_nc_cat=106&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=a1Tq6KI-VdUQ7kNvwGkjNM7&_nc_oc=AdqcnpqFJCtiRbNBGuS3gGo_kG-VYRWdHK7SnqzIIsN8f1eoWQgkeJEWRtve7MqI2hEFcQZaAxj89KgpdVJ027Rd&_nc_zt=14&_nc_ht=scontent.fdel27-8.fna&_nc_gid=8JW2Q1mCYFbKMdcyEX47vQ&_nc_ss=7b289&oh=00_AQB8vIpEH3vnDsuG3YHF2abemYbIt6lwSdV1MNBxGU7TXg&oe=6A607971)

### 3. Confirm Meta Business Account

You’ll need to connect a Meta Business Account to the Conversions API Gateway. To do so, click **Get Started** and follow the steps to connect your business using your Meta Business Account.

Click **Next**.

![Confirm Meta Business Account step showing a Connect your business to Facebook panel with a Get Started button.](https://scontent.fdel27-4.fna.fbcdn.net/v/t39.2365-6/386158645_859253885760876_8256905306547668779_n.png?_nc_cat=106&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=65btzsbKKUoQ7kNvwGx5yrv&_nc_oc=AdpyN11H6PhemW3AvLlesERtULSNSHpjyed99WejxI3KfcPtSjOOZiu5pCR4oymRurYuQJsSKFL-0Wpkxde2zwcD&_nc_zt=14&_nc_ht=scontent.fdel27-4.fna&_nc_gid=8JW2Q1mCYFbKMdcyEX47vQ&_nc_ss=7b289&oh=00_AQDy7WH6pchFg273E6LLd2U4ZPnfwPNuZCXF1yIoVgTEKA&oe=6A60711A)

Once you’ve completed the connection process, click **Next**.

### 4. Domain Management

Initial configuration is complete, and the final step in setting up this Conversions API Gateway instance is to add a domain.
