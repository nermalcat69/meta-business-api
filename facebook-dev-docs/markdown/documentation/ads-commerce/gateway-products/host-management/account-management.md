# Managing Hosted Accounts

Updated: Feb 2, 2025

Every admin or standard user for a host account can create and manage accounts in the **Manage accounts** menu.

*Best practice*: Create one account for each advertiser.

## Managed vs. Unmanaged Accounts

There are two ways to onboard an advertiser account based on how you want to manage your clients.

**Unmanaged account**: The business wants to manage their account by themselves and does not want any of the host users to have access to it. If this is the case, the host admin chooses to invite someone else to be account administrator when creating their account. None of the host users will have access to the account, admins included.

**Managed account**: The business wants the host to also manage their account and the host users to have access to it, according to their permissions. If this is the case, the host admin chooses to continue as account administrator when creating their account. All host users will have access to the account, including user management, data source management and event activity data, according to their permissions.

## Create a Managed Account

To create a new managed account click on **Add account** from the Manage accounts menu or the **Add** button in the menu.

First, provide the name of the account to be created and click **Continue**.

![Add new account Add an account name step with the Managed account name field and a Continue button.](https://scontent.fdel27-5.fna.fbcdn.net/v/t39.2365-6/475163833_2959205260909248_1946108326834107244_n.png?_nc_cat=111&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=lvQa66gktacQ7kNvwFoGD10&_nc_oc=AdrZtAzyDL9S75q50UxkopKxD4W6SJsv97gT5Ueg3_JuCGw8jIAhBolNh3b2grIWmGjF-Z1bBmBsPEr_I6w-nYNQ&_nc_zt=14&_nc_ht=scontent.fdel27-5.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQBm4GTnfxOZWvld37-jmcE8Dlsuya_HwqtfRK4KD-RHWw&oe=6A6072AC)

Select **Continue as account administrator** to create a Managed account. Click on **Next**.

![Choose an administrator step with Continue as account administrator selected to create a managed account.](https://scontent.fdel27-7.fna.fbcdn.net/v/t39.2365-6/475039093_1796148981144819_6270704799913293069_n.png?_nc_cat=105&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=sOnMsuR4zvgQ7kNvwEVKOgu&_nc_oc=AdruSuH9n8qrs--2SgB_duJFKLnNAYwYYceo867hEqYIhWTqO9POa9XRgMMTPv5frJ5RRvOsDNSwaAgqIxh6Ukcc&_nc_zt=14&_nc_ht=scontent.fdel27-7.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQB0ebooYG0qz9jxSjZjc7CD4ni4nvSDBJlJThgQD90xjw&oe=6A60A018)

Choose to use [manual advanced matching](https://developers.facebook.com/docs/meta-pixel/advanced/advanced-matching) if it is appropriate. This can be adjusted later.

## Change the Name of a Managed Account

Host admin or standard users can change the name of an account. To do so, click on the three dots button next to the account’s name and select Edit name.

![Edit Tenant Name dialog with a Tenant Name field and Cancel and Confirm buttons.](https://scontent.fdel27-2.fna.fbcdn.net/v/t39.2365-6/474838311_454973804217046_8586053608389663055_n.png?_nc_cat=101&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=6rmKFluo358Q7kNvwHFVgr7&_nc_oc=Adr1ss-4Nhk3SbB5KuXeR8XKIuLaAVB89hq5y0q-JEId4_FHG_PjtUL9jjShjghUnuJZp_RDHRjcP0pzcRwM5aFl&_nc_zt=14&_nc_ht=scontent.fdel27-2.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQAXGatFJ24-jDCmTsn9frx0DLbRf8SsnUbrI3kyhGjyMQ&oe=6A609EA9)

Provide the new name for the account and then **Confirm**.

## Deactivate a Managed Account

Host admin or standard users with appropriate permissions can deactivate an active managed account.

To deactivate a managed account select **Deactivate** from the three dots button next to the account name.

![Manage Accounts list with the three dots menu open on Account 1 showing Edit name, Deactivate, and Delete options.](https://scontent.fdel27-7.fna.fbcdn.net/v/t39.2365-6/475156368_498187509975054_4193630359953869217_n.png?_nc_cat=107&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=x_qwwxhHyDEQ7kNvwHINcWb&_nc_oc=AdqEVNu8nc9mvMfShIjIpQ-URI4eAwr-kRlQAH40hNAtC91yhebfRA7jSxzFh236POa6FF3kk6yVHJuK0zleoQC_&_nc_zt=14&_nc_ht=scontent.fdel27-7.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQATaUMWkygYPD267TJYPuE7gaH7liWPTkP2jDtCrirwFw&oe=6A607DEE)

A deactivated account will have a Deactivated status and a disabled **See details** button.

![Manage accounts list with Account 1 showing a Deactivated status and a disabled See details button.](https://scontent.fdel27-3.fna.fbcdn.net/v/t39.2365-6/474851439_9335784539818861_944369374000740801_n.png?_nc_cat=111&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=CxhQabXjIbgQ7kNvwGzXSAx&_nc_oc=AdpQlAkzeX8RP0vOTLuRUY-QNKcqwSgQL3odJEKvbTyh-s-X3G-J2yazKOp4UVzBYrQYrvNvPAxH3SSBVnohSp5d&_nc_zt=14&_nc_ht=scontent.fdel27-3.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQC67Fg1AvU-5JU5JOOJn4MHT2PVjQIYtL4s-sczn0Gagw&oe=6A60915A)

## Delete an Account

Host admin or standard users with sufficient permissions can delete both unmanaged and managed active accounts.

For all accounts, go to Manage accounts → the three dots button next to the account name and then Delete and Confirm.

![Manage accounts list of host accounts with the three dots menu open showing Edit name, Deactivate, and Delete options.](https://scontent.fdel27-1.fna.fbcdn.net/v/t39.2365-6/475139865_594430660065840_7178218627880266779_n.png?_nc_cat=100&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=fHZoPmwERyEQ7kNvwFn9ChP&_nc_oc=AdpV57vnm08GuhReb0Gmc1OPaOlOCnTPosHErarP6sBSSVFpMs8A9fxf5YyL_cDfFD9LWPtSwWqYoF4cvojLEP6-&_nc_zt=14&_nc_ht=scontent.fdel27-1.fna&_nc_gid=WKHHgAJpBkdX8o6TerwDeQ&_nc_ss=7b289&oh=00_AQC8Rp_Ta3NG6IBoHr3Nk50-GwZNnp7p_trOUXJsFU7H8w&oe=6A608513)

Deleting the account will remove all users and disconnect all data sources connected to the account.
