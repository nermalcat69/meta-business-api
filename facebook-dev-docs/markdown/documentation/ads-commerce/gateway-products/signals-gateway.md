# Uninstall the Conversions API Gateway

Updated: Feb 2, 2025

### Step 1 - Go to the See Details Page from Events Manager

This step will disconnect an onboarded Pixel from the Conversions API Gateway.

Navigate to the connection using **Events Manager -> Data Sources -> Data Source settings -> Conversions API**. To start disconnecting from the UI, click on the **See Details** button:

Repeat this step for each data source connected to the Conversions API Gateway.

### Step 4 - Delete Cloudformation Stack

This step will delete AWS resources created by the Cloudformation stack, including EC2 instances and security groups. Delete the stack by clicking the **“Delete”** button.

![AWS CloudFormation Stack info page for the Conversions-API-Gateway stack with the Delete button highlighted.](https://scontent.fdel27-5.fna.fbcdn.net/v/t39.2365-6/355682778_235269665947799_355364010111447473_n.png?_nc_cat=102&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=fL13JkGUXj0Q7kNvwFQvXfM&_nc_oc=Ado9mytMiRhTqHOuAIj1f40wIQI7M1Jm58Bl1o7hKXazO8qLgGIate2mMOrKnkEcZxnqMTq0S0ju4IFcGZQxpDKw&_nc_zt=14&_nc_ht=scontent.fdel27-5.fna&_nc_gid=45oafa_P3MV_jI14xQ9Uuw&_nc_ss=7b289&oh=00_AQCpcpO85TzkKnusMgvvPZNHLLkT4O4frYe_TSwSharfXA&oe=6A608317)

### Step 5 - Delete Log Group

The log group is generated by the Conversions API Gateway with permission you granted when you were installing the Conversions API Gateway.
Navigate to **CloudWatch** -> **Log Groups**, find the log group by searching for “conversions-api-gateway”, select it and then delete it by **Actions** -> **Delete log group(s)**

![AWS CloudWatch Log groups list filtered for conversions-api-gateway with a log group selected and the Actions menu open on Delete log group(s).](https://scontent.fdel27-1.fna.fbcdn.net/v/t39.2365-6/275215867_929448787740994_8602858112847270985_n.png?_nc_cat=107&_nc_map=urlgen_bucketless&ccb=1-7&_nc_sid=e280be&_nc_ohc=hlGldhS3LRUQ7kNvwEsCnyy&_nc_oc=AdpoDNc7BW8R9xRfAgtKO17mSMiD_oKW2optC0PA7dYR4ml6QN69so8yPB6iGKvJ1y38UveK8_Upgm16BSe_ZUKH&_nc_zt=14&_nc_ht=scontent.fdel27-1.fna&_nc_gid=45oafa_P3MV_jI14xQ9Uuw&_nc_ss=7b289&oh=00_AQAc1UUh-AD-9GBY9drU-GBc4Xap_Wlh7uccrSmxl3vuWA&oe=6A60738F)
