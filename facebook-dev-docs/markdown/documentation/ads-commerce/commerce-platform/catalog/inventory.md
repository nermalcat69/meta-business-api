# Flexify Integration Guide

Updated: Jul 22, 2021

* Install the Flexify app

* On Flexify app page, click on **Add additional product data for Facebook Shops**

* Select the relevant categories of the products in your Shop.
  
* For each category, follow the steps for tagging your products and adding category attributes in bulk.
  
* Either add tag individually to products or use Shopify's bulk product editor.
  
* Continue in Flexify's bulk editor for the category selected to add additional information in bulk to products of that category.

Repeat steps 2-4 for the remaining categories in your Shopify Catalog
