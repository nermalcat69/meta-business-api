# Ad Account User

Updated: Dec 4, 2025

## Reading

Data of App Scoped User.

```
curl GET \
https://graph.facebook.com/v25.0/act_<AD_ACCOUNT_ID>?fields={fieldname_of_type_AdAccountUser}
```

#### Parameters

This endpoint doesn't have any parameters.

#### Fields

| Field | Description |
| --- | --- |
| `id` *numeric string* | ID of the App Scoped User |
| `name` *string* | User public full name    default |
| `tasks` *list<string>* | Tasks of App Scoped User    default |

## Creating

You can't perform this operation on this endpoint.

## Updating

You can't perform this operation on this endpoint.

## Deleting

You can't perform this operation on this endpoint.
