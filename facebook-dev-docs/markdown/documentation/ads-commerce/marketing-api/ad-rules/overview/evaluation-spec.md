# Ad Rules Specs

Updated: Jun 21, 2026

Ad rule specs determine which actions to take and which objects those actions apply to.

### [Evaluation spec](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-rules/overview/evaluation-spec)

An ad rule evaluation spec determines the objects upon which the rule executes its action.

### [Execution spec](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-rules/overview/execution-spec)

An ad rule execution spec determines the action that applies to all objects that pass evaluation.

### [Change spec](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-rules/overview/change-spec)

The change spec applies to execution types such as `CHANGE_BUDGET` and `CHANGE_BID`.
