# Facebook Video API from Meta

Updated: Jun 4, 2024

The Facebook Video API from Meta allows apps to publish and crosspost videos on Facebook Pages administered by app users and obtain insights for these videos.

## Common Uses

* [Publishing Videos on Pages](https://developers.facebook.com/documentation/video-api/guides/publishing)
* [Crossposting Videos](https://developers.facebook.com/documentation/video-api/guides/crossposting)
* [Getting insights on published Videos](https://developers.facebook.com/documentation/video-api/guides/insights)
