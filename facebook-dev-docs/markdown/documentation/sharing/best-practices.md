# Webmaster Tool

The [Webmaster Tool](https://developers.facebook.com/webmaster) allows you to see how often Facebook has crawled your [verified domains](https://developers.facebook.com/documentation/sharing/domain-verification) and how often they have been shared on Facebook. It also allows you to add to the allow list domains that are able to specify your verified domains as their [canonical URLs](https://developers.facebook.com/documentation/sharing/webmasters/getting-started/versioned-link).

[Webmaster Tool](https://developers.facebook.com/webmaster)
