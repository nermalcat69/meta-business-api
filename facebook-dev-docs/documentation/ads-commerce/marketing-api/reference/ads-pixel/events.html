Creating

/{ads_pixel_id}/events

You can make a POST request to events edge from the following paths:
  • /{ads_pixel_id}/events
When posting to this edge, no Graph object will be created.

Example

POST /v25.0/<PIXEL_ID>/events HTTP/1.1
