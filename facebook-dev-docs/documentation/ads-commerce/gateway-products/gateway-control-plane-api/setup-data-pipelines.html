Conversions API Gateway and Signals Gateway Control Plane API: Reference

Conversions API Gateway and Signals Gateway Control Plane API: Reference

Updated: Jun 18, 2025
Starting from Conversions API Gateway and Signals Gateway v2.2.0, up-to-date versions of the Control Plane API reference docs, including examples with sample data, can be accessed inside your gateway UI. To find these docs:
  • Click on Settings
  • Choose API accounts
  • Click the API Reference link at the top of the API accounts page

Role

Field Description
name
String
Full name of the role.
Roles can be for agency (partner) users or advertisers (account) users.
Role name follows the following format:
For partner: agency-[ROLE-ACTION]
For advertisers: advertiser-[ROLE-ACTION]-[tenantId]
[ROLE-ACTION] -- This describes the responsibility of the role. It can have one of the 3 values:
  1. admin
  2. user
  3. view
[tenantId] - The id of the account whose roles are being assigned to users
displayName
String
This is the [ROLE-ACTION] of the role
tenantName
String
Name of the tenant/account.

UserType

Value Description
AGENCY
This is for partners/host
ADVERTISER
This is for tenant/account.
