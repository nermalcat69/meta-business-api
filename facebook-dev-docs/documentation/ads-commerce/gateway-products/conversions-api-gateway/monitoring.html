Include Facebook Login Data in the Conversions API Gateway

Updated: Feb 2, 2025
Including customer Facebook Login for advanced matching in the Conversions API is available from v1.6.0. This feature can be helpful for improving the Event Match Quality⁠ Score of events sent from websites that use “Login with Facebook”.
The feature is disabled by default in the Conversions API Gateway but advertisers can turn it on by following the below steps.

How to Enable the Feature

1) Go to your Conversions API Gateway admin console page, which is hosted on your subdomain.
Note: You can also launch your Conversions API Gateway instance in the Events Manager Setting Tab. Look for Conversions API Gateway section and select Launch.

2) Log in to your Conversions API Gateway admin console.
3) In Overview Page, go to Advanced matching Card and look for option: Include customer Facebook Login ID for advanced matching.

4) Enable toggle for Include customer Facebook Login ID for advanced matching.
