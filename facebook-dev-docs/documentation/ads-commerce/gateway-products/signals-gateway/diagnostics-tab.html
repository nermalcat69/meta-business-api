Messages Page

Updated: Feb 2, 2025
The Messages page lets you view system messages relevant to your account, such as diagnostics info. These messages are refreshed daily.
If an asset, such as a data source, has multiple issues or messages associated with it, the issue with the highest level of severity will be displayed. Messages can be filtered by severity using the tabs at the top of the table.
Messages under the “All” tab are sorted by severity, with the most severe at the top. To see more about an issue, select See details for additional information. You can also search messages using the search bar by selecting a single account (or if there’s only one account available).
In the example below, the account has two diagnostic messages, one for the “My Pixel” data source and one for the “My App” data source.


Clicking the “…” button and selecting See details will direct you to the Diagnostics Tab of the related data source (in this case it would be “My Pixel”).
Additionally, if you have access to multiple accounts, all of the messages for these accounts will be displayed on the Messages page. For systems that host multiple accounts,there is an additional Account filter and an Account column in the table.
