Plugins

Updated: Jul 10, 2025
The plugins page is a powerful tool that allows instance administrators to configure the plugins used in their instance. This page provides a centralized location for managing plugin settings, ensuring that your instance is optimized for your specific needs.
As an instance admin, you have the ability to modify the plugin settings on this page. If any issues arise during the modification process, a warning will be displayed along with instructions on how to resolve the issue.
The following plugins are available:

1. Meta Conversions API plugin

  • This plugin allows you to add the Meta Conversions API as a data destination for your data pipelines in the Signals Gateway.

2. Google BigQuery plugin

  • This plugin allows you to add Google BigQuery as a data destination for your Signals Gateway pipelines.
