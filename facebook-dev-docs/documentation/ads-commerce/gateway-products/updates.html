Conversions API Gateway and Signals Gateway: AWS ECS Express Cost Monitoring

Conversions API Gateway and Signals Gateway: AWS ECS Express Cost Monitoring

Updated: Jan 23, 2026
Follow the steps below to set a budget and cost alerts for your Conversions API Gateway and Signals Gateway connection.

Set a Budget

After connecting to your Meta business portfolio, launch the Cost management page on the left sidebar under Settings.
Note that all amounts shown on this page are in US dollars.

View Current Cost


On the Cost management page, you can view your current cost and your current maximum budget for the account.

View Cost History


This page will also show the previous month’s costs and an average monthly cost, based on the previous six months.

Update Budget Limit

In the Cost management page, use the Edit button next to the current budget set for the account to edit the budget limit.

In the screen that opens, enter the maximum budget for the account. Click Continue to save your changes.

Update Alerts

Update alert thresholds

In the Cost management page, click the Edit alerts link next to the current budget for the account.

In the screen that opens, enter the percentage(s) of budget usage that should trigger an alert. For example, when 50% of your budget has been spent, you can choose to receive an alert. Click the Add a new alert button to add another budget percentage threshold, or click the trash icon to remove one. You can set up a maximum of two alerts.

Click the Continue button in the modal to save your changes.

Update Alert Email Recipients

In the same modal, there is a section at the bottom where you can view and edit the emails of account users that should be notified when an alert threshold is reached.
Click the Add a new recipient button to add a new email address to the list of recipients. Click the trash button to remove the recipient.
