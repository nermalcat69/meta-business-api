Get Started

Updated: Apr 28, 2026
The first step in accessing the Commerce API is creating a test commerce account. You can create a commerce test account using pre-filled test banking and tax information, manage test inventory, and place test orders using test credit cards.
Note: Test commerce accounts are only available for sellers in the US.

How it Works

  • Test accounts don’t process payouts or charge credit cards in checkout. Once created, a test account cannot be converted to a regular account.
  • You can test API calls in your test accounts without incurring payment or payout.
  • You can only create a limited number of test accounts under one Business Manager. Some features of regular commerce accounts are not available in test accounts.
  • You can omit the following information and can place test orders in the account:
    • No bank account and routing number needed
    • No tax and payment information needed
    • No real product catalog needed
Instagram channel is not yet available in test accounts. Test with orders from your Facebook Shop while building the integration, as the API that powers both platforms is the same.

Step 1: Create a New Test Account

Test accounts are designed to quickly ramp up on the Facebook Commerce platform. It’s a great way for you to learn how to set up Commerce Manager and start building an integration with your ecommerce platform or order management system (OMS):
  1. Go to Commerce Manager⁠ and click Create a Test Shop.
  2. Follow the prompts to complete the setup.

Step 2: Add Products

After setting up the test account, you are returned to your account’s home page.
  1. Learn how to navigate Commerce Manager⁠.
  2. Under the Catalog tab, add some products⁠, manually or by uploading a product feed file.

Step 3: Place Test Orders

Now that the products are added, you can visit a Shop on the Test Page you created and connected to your Test Account during the setup and place test orders.
  1. Go to the Facebook Page connected to your test commerce account.
  2. Under the Shop tab, you see your newly added products and can now place test orders with pre-filled credit card information.
A Shop attached to a test commerce account is only visible to the admins of the page.

Step 4: Manage Orders

Once you place a test order, it’s visible in your Commerce Manager, where you can find and manage orders⁠ placed in your test Page Shop:
  • List orders
  • Mark orders as shipped, provide a tracking number
  • Refund orders
  • Cancel orders
