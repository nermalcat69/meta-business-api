# Requirements

Updated: Mar 28, 2024

Requirements for ads on Instagram are similar to those for Facebook, but there are exceptions. These pages help you navigate through the differences.

### [Media Requirements](https://developers.facebook.com/documentation/ads-commerce/instagram/ads-api/reference/media-requirements)

Ad creative requirements are different on Instagram. Learn more about recommended aspect ratio, size, and cropping. Also see, [Image Crops](https://developers.facebook.com/documentation/ads-commerce/marketing-api/image-crops).

### [Data and Call to Action Requirements](https://developers.facebook.com/documentation/ads-commerce/instagram/ads-api/reference/data-cta-requirements)

Create an ad creative using in-line creation on `object_story_spec` and `call_to_action`.
