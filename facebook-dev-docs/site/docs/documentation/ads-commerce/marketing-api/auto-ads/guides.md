# Get started

Updated: Dec 5, 2025

This guide explains how to set up your catalog for automotive ads.

## Before you start

To create automotive ads, you need a [Facebook Page⁠](https://www.facebook.com/pages/create/) and an [ad account⁠](https://www.facebook.com/ads/manager/accounts).

## Step 1: Create your automotive catalog

To create an [automotive catalog](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/catalog), connect a data feed or upload data to Meta servers. The data should contain all the required fields for the vehicles that you want to advertise. The [Reference](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/reference) lists the required fields.

## Step 2: Set up automotive events on your website

Automotive ads use [standard and required events](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/events), and every event has a set of parameters. You can use the Meta Pixel on your website and mobile app events in your Android app and iOS app.

## Step 3: Build your automotive audience

To successfully [build and manage your audience](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/audience-mgmt), you can:

* [Associate your Pixel with your catalog](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/audience-mgmt#associate-pixel)
* [Retarget an audience](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/audience-mgmt#retarget)

## Step 4: Create and start your automotive ads

This step assumes you have a catalog and product set ready, and have set up the required vehicle events on your website or in your mobile app. If not, see [catalog setup](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/catalog) and [event setup](https://developers.facebook.com/documentation/ads-commerce/marketing-api/auto-ads/guides/events).
