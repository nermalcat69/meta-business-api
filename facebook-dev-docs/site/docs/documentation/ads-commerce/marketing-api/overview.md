# Marketing API

The Marketing API is a collection of [Graph API](https://developers.facebook.com/docs/graph-api) endpoints and other features that can be used to help you advertise across Meta technologies. Before advertising on Facebook, Instagram, Messenger, and WhatsApp, we recommend you learn about Meta's [ad campaign structure](https://developers.facebook.com/documentation/ads-commerce/marketing-api/overview#how-it-works) to understand the objects you're working with and how they relate to each other.

[Get started](https://developers.facebook.com/documentation/ads-commerce/marketing-api/overview#how-it-works)

## Get Started

Begin creating and managing Meta ad campaigns by following step-by-step guidance on setting up campaigns, ad sets, and creatives using the Marketing API

Basic Ad Creation

Get detailed guidance on how to set up campaigns, ad sets, and ad creatives, including code samples that illustrate the implementation process.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/marketing-api/get-started/basic-ad-creation)

Manage Campaigns

Learn key operations you can perform using the Marketing API, including how to modify, pause and delete campaigns.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/marketing-api/get-started/manage-campaigns)

Ad Optimization Basics

Use Marketing API endpoints that serve as essential tools for developers to manage audiences and analyze ad campaign insights.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/marketing-api/get-started/ad-optimization-basics)

## Related APIs

Explore additional APIs to enhance ad targeting, manage catalogs, maintain business presence, and integrate commerce tools across Meta technologies

Conversions API

Connect the marketing data on your servers to the Meta systems that optimize ad targeting, decrease cost per action, and measure results.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/conversions-api)

Catalog API

Create a catalog of items you want to promote and use it to run ads, sell from a shop on Facebook or Instagram, and more.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/catalog)

Business Management API

Create and maintain your business's organic and paid presence on Facebook, Instagram, Messenger, and WhatsApp.

[Learn more](https://developers.facebook.com/docs/business-management-apis)

Commerce API

Enables businesses to integrate their infrastructure with the tools available to sell products across Meta technologies, including shops and Marketplace.

[Learn more](https://developers.facebook.com/documentation/ads-commerce/commerce-platform)
