# Supported partnership ads configurations

Updated: Jun 30, 2026

The Instagram Partnership Ads API supports the following partnership ad configurations. Each linked page covers the prerequisites, parameters, and examples for that configuration.

### [Placement asset customization](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/placement-asset-customization)

Boost existing Instagram posts as partnership ads with Placement asset customization.

### [Advantage+ creative](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/advantage-creative)

Boost existing Instagram posts as partnership ads with Advantage+ creative.

### [Click to message destinations](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/click-to-message-destinations)

Create partnership ads that click to different messaging destinations.

### [Partnership ads with Advantage+ catalog ads](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/advantage-catalog-ads)

Create partnership ads using Advantage+ catalog ads with either a carousel or collection format.

### [Testimonials in partnership ads](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/testimonial-ads)

Add testimonials from partners about their experience with your product or service.

### [Lead generation for partnership ads](https://developers.facebook.com/documentation/ads-commerce/marketing-api/ad-creative/partnership-ads/ads-creation/supported-configurations/lead-ads)

Gather contact information from people who might be interested in your product or service.
