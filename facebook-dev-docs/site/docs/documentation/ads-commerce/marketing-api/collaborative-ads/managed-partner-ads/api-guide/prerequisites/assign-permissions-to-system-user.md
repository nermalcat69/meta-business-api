# Create an admin system user

Updated: Jun 21, 2026

This page has guidance on how to create an admin system user for Managed Partner Ads. An admin system user can create new users, add accounts, assign permissions, and access all assets belonging to the business.

## Before you begin

You must own a Meta app that's associated with your Meta Business Suite to add system users. See [Add Apps to Your Meta Business Suite⁠](https://www.facebook.com/business/help/2199735813629697) for more information.

## Create the system user

* Go to the [Business Settings⁠](https://business.facebook.com/settings).
* Under **Users**, click **System Users**.
* Click the **Add** button.
* Name the system user and select **Admin** as the system user role.
* Click **Create system user** to complete the setup.
* After you create the system user, click the **Edit** button and grant the [**Finance Editor**⁠](https://www.facebook.com/business/help/442345745885606) role.

You can only have ***one*** system admin user per Meta Business Suite account.
