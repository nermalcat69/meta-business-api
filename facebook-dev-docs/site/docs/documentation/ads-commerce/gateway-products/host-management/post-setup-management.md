# Conversions API Gateway and Signals Gateway: Post-Setup Management

Updated: Feb 2, 2025

## Automatic Updates

Automatic software updates are only available with version 1.0.8 and newer for Conversions API Gateway and Signals Gateway and are turned on by default. If automatic software updates are turned on, the latest security and performance optimizations will be installed automatically as soon as they become available. Admins can turn automatic updates off at any time by changing the toggle in the Updates menu under the Settings tab.

If automatic software updates are turned off, the business is responsible for updating the “Gateway Product” Conversions API Gateway or Signals Gateway manually using the Updates menu under the Settings tab.

## Automatic System Optimizations

Automatic System Optimizations are available starting with installations of version 1.11.0 and newer for Conversions API Gateway, and Signals Gateway, and are turned on by default. If it is turned on, this setting provides support for your Gateway infrastructure lifecycle management to automatically keep the instance on the latest optimized infrastructure.

## Hardware and Operating System

Advertisers are responsible for the hardware and operating system infrastructure maintenance of the third-party cloud provider (for example, AWS) when hosting a Gateway Product.
