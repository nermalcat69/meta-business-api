# Onboarding and Offboarding Unmanaged Hosted Accounts

Updated: Feb 2, 2025

Before onboarding a new account, check that you have the necessary prerequisites.

To begin, the invited account admin will receive an invitation link/email, and will need to create their admin user profile. The invited account admin must open the link, provide the email address that received the invitation, and set a password. By clicking **Submit**, they will become the account admin and first account user. The account admin can then add other account users, manage data sources, and update account settings.

It is strongly recommended that you set up data routing and enable a first-party connection between the pixel and the Conversions API Gateway or Signals Gateway.

An account admin or standard user can delete the account by selecting **Delete account** in the Account settings menu and then Delete account.
