# Data Source Management

Updated: Feb 2, 2025

When creating a data pipeline, you can create or add data sources that will receive inbound events to the data pipeline. Data sources can also be created from the data pipeline details page after the data pipeline is created, from the data pipeline details page. Once a data destination is created, you can access the data pipeline details page to perform various data source operations such as creating additional data source types, and updating or deleting existing data sources.

## Creating a Data Source

Click the “+” button in the pipeline details page. If the “+” button is not available or disabled, it means that all supported data source types have already been added to the data pipeline.

After clicking the “+” button, there will be a default name for your data source, but you can modify it if needed. Then, follow the provided instructions to complete the data source creation process.

## Deleting a Data Source

To delete a data source, click the data source in the data pipeline details page, then find the **Actions** drop down button on the top right corner, and click **Remove**.

## Editing a Data Source

You can update a data source’s name, or its configurations.

To update the data source name, click the data source, and click the **Edit** sign next to its current name.

To update the data source’s configurations, click the **Configurations** tab in the data source details page.
