# Include Facebook Login Data in Conversions API Gateway Pipelines

Updated: Feb 2, 2025

Signals Gateway advertisers who have built Conversions API Gateway pipelines can use Facebook Login data. This feature can be helpful for improving the [Event Match Quality⁠](https://www.facebook.com/business/help/765081237991954?id=818859032317965) Score of events sent from websites that use “[Login with Facebook](https://developers.facebook.com/documentation/facebook-login/web)”.

The feature is disabled by default in the Signals Gateway but advertisers can turn it on by following the below steps.

## How to Enable the Feature

* Go to your Signals Gateway admin console page, which is hosted on your subdomain.

**Note**: You can also launch your Signals Gateway instance in the Events Manager Setting Tab. Look for the Signals Gateway section and select Launch.
