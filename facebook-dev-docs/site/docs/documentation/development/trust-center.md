# Support

Updated: Jun 30, 2026

You can access support resources by clicking the **Support** tab, located on the upper-right navigation menu of any Meta for Developers page. ## Direct Support

[Developer support](https://developers.facebook.com/support) is now available for developers who need assistance while completing a Data Access Renewal assessment and throughout their experience on Meta for Developers. Support is provided for Data Protection questions, data handling questions, and app review questions.

### What to expect

* A functionality to submit a question to direct support and get a response within 48 hours
* An overview of [Data Access Renewal](https://developers.facebook.com/documentation/resp-plat-initiatives/data-access-renewal) and [Data Handling Questions](https://developers.facebook.com/docs/resp-plat-initiatives/data-access-renewal/tutorial/data-handling-questions)
* Answers to [frequently asked questions](https://developers.facebook.com/documentation/resp-plat-initiatives/data-access-renewal/faqs), with an option to input a question using free form text

### How to access

Support is available by directly asking a question using the “Ask a question” feature, or by selecting “Contact Direct Support”.

Here are the steps to get help from your developer dashboard. Note that there are two main ways of accessing the Help page, depending on where you are.

* **Within developers.facebook.com/support:**

If you are on developers.facebook.com, you can access help via the “Get Help” on the bottom right.

## Contact Us

* [Developer Appeals](https://developers.facebook.com/appeal/) – View app appeals, appeal a descision, restore apps, and get help finding apps you used to own but can not find.
* [Report a Bug](https://developers.facebook.com/support/bugs/) - Report a platform bug, view other bugs that have been reported, and see the status of a bug report.
* [Report a Platform Data Incident](https://developers.facebook.com/incident/report/) – Report security incidents involving platform data or systems that process plaform data, or platform data that was received in error or processed in violation of Facebook policies and terms.
* [Report a Platform Policy Violation⁠](https://www.facebook.com/help/117257561692875) – Report an app or game for a policy violation in the Facebook Help Center.
* [Business Manager⁠](https://business.facebook.com/direct-support/) – Visit the Business Manager for any questions about business issues in including payments.
