# Social Plugins

Updated: Mar 27, 2026

This document lists the Facebook Social Plugins available that allow your app to share and embed content.

## Documentation Contents

When using Social Plugins, follow the [Best Practices](https://developers.facebook.com/documentation/plugins/best-practices).

### [Share Button](https://developers.facebook.com/documentation/plugins/share-button)

Share content to Facebook, with friends, with a group, or in a private message.

### [Page Plugin](https://developers.facebook.com/documentation/plugins/page-plugin)

Embed components of a Facebook Page onto a webpage.

### [Embedded Posts](https://developers.facebook.com/documentation/plugins/embedded-posts)

Add public posts into the content of a webpage.

### [Embedded Video & Live Video Player](https://developers.facebook.com/docs/plugins/embedded-video-player)

Add public Facebook videos and Facebook live videos to a webpage.

## FAQ

**Will this break my website?**

No. The plugins will gracefully degrade to invisible elements, this should not otherwise impact your website.

**Do I need to remove the plugin code?**

No, we recommend removing the plugin code for a cleaner user experience, but this is optional.

**Are there replacement plugins?**

No replacement plugins are being provided. This change reflects evolving web development practices.

**How will I know if my app is affected?**

We will send two developer alerts to all applications that have used these plugins recently: the first will provide the initial 3-month announcement, and the second will be sent one week before the discontinuation date. Please ensure your developer account email is up to date to receive these important notifications.

**Can I get an extension on the timeline?**

The Feburary 10, 2026 date is final to allow us to focus resources on modern platform features.

**What if I need help deciding what to do?**

Contact our [Developer Support team](https://developers.facebook.com/support) or post in our [Community Forum](https://developers.facebook.com/community) for guidance.
