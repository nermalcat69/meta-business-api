# Monetize Instant Games with Audience Network

Updated: Aug 18, 2023

The goal of this article is to provide you with the most efficient path to onboard or activate an Instant Games version of your game in Monetization Manager - whatever your current mobile app setup.

Once activated, you can monetise both the mobile app version and the Instant Games version of your game with Audience Network.

## Prerequisites

This article assumes that:

* You have applied to become an [approved Partner⁠](https://www.facebook.com/fbgaminghome/developers/instant-games/approved-partner-program), and been accepted.
* You have [registered your gaming app](https://developers.facebook.com/documentation/development) with Meta to obtain a Meta app ID.
* You have [implemented Instant Games in your app](https://developers.facebook.com/docs/games/onboarding/instant-games/get-started#app-setup)
* You have created a Meta Business Manager and are able to login to the [Meta Monetization Manager⁠](https://www.facebook.com/audiencenetwork/getting-started).

## Adding Features

To enable Facebook Instant Games to use social features, you will need to submit each permission for an app Review. We strongly recommend that you review which features you will need in advance, in order to avoid a delay in launching the game.

The following links explain some of the feature options:

* [In-app Purchases](https://developers.facebook.com/docs/games/monetization/in-app-purchases) - information about how to implement in-app purchases in your Facebook game.
* [Organic Acquisition](https://developers.facebook.com/docs/games/organic-acquisition) - how to leverage Facebook platforms to attract and retain players.
* [Sharing](https://developers.facebook.com/docs/games/social-and-retention/features/sharing) - use sharing APIs to help players share key moments or achievements to their timeline, and into their friends' Feeds.
* [Invite and Player Finder](https://developers.facebook.com/docs/games/social-and-retention/features/play-with-friends/player-finder) - enables users to find friends to share their gaming experience with.
* [Invites](https://developers.facebook.com/docs/games/social-and-retention/features/invites) - help players invite friends to play with them.
* [Paid Acquisition](https://developers.facebook.com/docs/games/paid-acquisition) - how to attract more players to your Facebook game.
* [Click-to-play ads](https://developers.facebook.com/docs/games/player-acquisition/best-practices) - an outline of recommended best practices for Instant Game ad campaigns.

## Audience Network Review

Once you have successfully set up the Instant Games version of your game with Audience Network, you'll see the Games platform card in your Monetization Manager. Click **Add payment account** to add payment information. This automatically starts the Audience Network review process.

## Integration

These resources will help you with your next steps when monetizing with Audience Network:

* See [In-app Ads for Instant Games](https://developers.facebook.com/docs/games/monetization/in-app-ads/instant-games) for instructions on Rewarded Video and Interstitial ads, including Sample Code.
* See [Banner Ads](https://developers.facebook.com/docs/games/monetization/banner-ads) for instructions on banner ads.

## Testing

* Once you integrate the ads, check out [Testing, Publishing and Sharing an Instant Game](https://developers.facebook.com/docs/games/onboarding/instant-games/get-started/test-publish-share).
* Test mode is automatically on in Audience Network when a game has not passed app Review, and you are using an [Admin or Dev role](https://developers.facebook.com/documentation/development/build-and-test/app-roles) to test it.
  Alternatively, you can follow Audience Network's [Test Your Implementation on the Platform](https://developers.facebook.com/documentation/audience-network/setting-up/testing/platform) article to test more proactively.

## App Review

Your game must pass both Audience Network review and [Meta App Review](https://developers.facebook.com/docs/app-review) before you can start monetizing.
