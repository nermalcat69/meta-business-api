# Terms and Policies

Updated: Oct 20, 2021

* [California Consumer Protection Act (CCPA)](https://developers.facebook.com/documentation/meta-pixel/implementation/data-processing-options) – Businesses operating in California are responsible for implementing code to comply with CCPA.
* [Facebook Cookie Policy⁠](https://www.facebook.com/policies/cookies/) – See our Cookie Policy for details about the cookies used and the data received.
* [General Data Protection Regulation (GDPR)](https://developers.facebook.com/documentation/meta-pixel/implementation/gdpr) – It applies to companies (regardless of where they are based) who process personal data about individuals in the European Union.
