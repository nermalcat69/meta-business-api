# Meta / Facebook Developer Docs Mirror

A crawlable Markdown mirror of Meta's developer documentation
(`developers.facebook.com`), built from a static HTML snapshot. Every page
links back to its original Facebook URL at the top of the article.

Browse by category below, or use search (top bar) to find a specific page.
